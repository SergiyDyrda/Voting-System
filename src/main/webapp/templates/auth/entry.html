   Java Enterprise проект с регистрацией/авторизацией и интерфейсом на основе ролей (USER, ADMIN). Система проведения голосованния для принятия решения, о месте проведения ланча. Администратор производит запись в систему данных о предложенных ресторанах (наименование, меню). Меню меняеться каждый день (администратор осуществляет обновления). Пользователи могут голосовать за ресторан, в котором они желают провести ланч. Каждый пользователь способен отдать только один голос. Если пользователь повторно голосует в тот же день:
  • Если голосование происходит раньше 11:00, предполагается, что пользователь изменил свое решение
  • Если голосование происходит позже 11:00, голос пользователя не учитывается по причине истечения срока допускаемого для голосования
